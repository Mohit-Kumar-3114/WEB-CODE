
Question- What are functions?
Answer- A JavaScript function is a block of code designed to perform a particular task. A JavaScript function is 
executed when "something" invokes it (calls it).




Question- What are callback functions?
Answer- Any function that is passed as an argument to another function so that it can be executed in that other 
function is called as a callback function.




Question- What is setTimeout function?
Answer- The setTimeout() function executes a function once after a delay. It takes two arguments: the first is the 
callback function to be executed, and the second is the time delay in milliseconds.





Question- What is setInterval function?
Answer- The setInterval() function executes a function repeatedly after a specified interval. It takes two arguments: 
the first is the callback function to be executed, and the second is the time delay in milliseconds after which 
callback is called again.
